# discord Clone
> Simple chatapp site built with Node.js, React, Redux, Express, MongoDB, Pusher
### Install Dependencies (Frontend)

```
cd discord
npm i
```
### Install Dependencies (Backend)

```
cd discordBackend
npm i
```
### run backend

```
cd discordBackend
npm test
```
### run frontend

```
cd discord
npm start
```
